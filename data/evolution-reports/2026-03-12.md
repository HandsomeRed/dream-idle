## ⚠️ 遇到的问题和解决

### 问题 1: Jest 配置 JSX 报错
**现象：** `Cannot use JSX unless the '--jsx' flag is provided`

**解决：**
```typescript
// jest.config.ts
transform: {
  '^.+\\.tsx?$': ['ts-jest', {
    tsconfig: {
      jsx: 'react-jsx',
      esModuleInterop: true,
    },
  }],
},
```

### 问题 2: toBeInTheDocument 找不到类型
**现象：** `Property 'toBeInTheDocument' does not exist on type 'JestMatchers'`

**解决：**
- 安装 `@testing-library/jest-dom`
- 在 `jest.setup.ts` 中导入
- 在 tsconfig 中添加 types 配置
