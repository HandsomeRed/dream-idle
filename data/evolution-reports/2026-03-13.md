## 💡 犯的错误和解决方法

### 错误 1：时区判断错误
**问题：** 股票监控误判为周末模式  
**原因：** 代码中强行 +13 小时转换北京时间，但系统已是 CST 时区  
**解决：** 移除时区转换，直接使用本地时间  
**固化：** 时区处理前先检查系统时区 `date` 命令

### 错误 2：手机访问空白页面
**问题：** 开发模式 HMR 不兼容手机浏览器  
**原因：** Vite 开发服务器使用 React Refresh 和 ESM 动态加载  
**解决：** 构建生产版本 (`npm run build`) + Python HTTP 服务器  
**固化：** 外网访问优先使用生产模式

### 错误 3：TypeScript 类型导入错误
**问题：** `verbatimModuleSyntax` 导致类型导入失败  
**原因：** 类型必须用 `import type` 语法  
**解决：** 修改 tsconfig 关闭严格检查 + 修复导入语句  
**固化：** 类型导入统一使用 `import type`
