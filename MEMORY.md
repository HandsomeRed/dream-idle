## 🎮 游戏设计原则

### 放置游戏核心要素
1. 反馈循环 - 立即反馈激励玩家
2. 增量进展 - 不断增加收益速率
3. 简单控制 - 任何玩家都能上手
4. 自动化收益 - 离线也能获得进度

- 回合制 → 自动战斗
- 任务 → 自动寻路 + 离线收益
- 经济 → 自动产出 + 离线收益
- 社交 → 异步交互 + 放置排行榜
