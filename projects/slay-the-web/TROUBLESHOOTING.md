# 🔧 故障排查指南

## 问题：无法访问 http://49.232.215.84:4001

### ✅ 服务状态检查

服务已正常启动：
- 后端：http://localhost:4000 ✅
- 前端：http://localhost:4001 ✅

### 🔍 可能的原因

#### 1. 云服务器安全组未开放端口

**解决方法：** 登录云服务器控制台，添加入站规则：
- 端口范围：4001
- 协议：TCP
- 来源：0.0.0.0/0

#### 2. 防火墙阻止

**检查命令：**
```bash
# 查看防火墙状态
ufw status

# 如果启用了防火墙，开放端口
ufw allow 4001/tcp
ufw reload
```

#### 3. iptables 规则

**检查命令：**
```bash
iptables -L -n | grep 4001
```

### 📝 临时解决方案

如果你现在想立即测试，可以用以下方式：

**方法 1: SSH 隧道**
```bash
# 在本地电脑执行
ssh -L 4001:localhost:4001 root@49.232.215.84
# 然后访问 http://localhost:4001
```

**方法 2: 使用已有端口**

你的梦幻放置在 :3000，可以让 Slay the Web 也用 3000 端口（需要先停止 DreamIdle）

或者使用 Nginx 反向代理（如果你有的话）

访问测试页面：http://49.232.215.84:4001/test.html

如果这个也打不开，说明是网络/防火墙问题。
