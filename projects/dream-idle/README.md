## 🎮 游戏简介

梦幻西游题材放置挂机网页游戏，经典回合制战斗的放置玩法改编。

**核心玩法：**
- 🗺️ 主线推图（10 章 100 关）
- ⚔️ 回合制战斗
- 📈 角色养成（升级/装备/技能）
- ⏰ 离线收益
