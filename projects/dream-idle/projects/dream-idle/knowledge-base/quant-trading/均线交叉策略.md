## 📖 核心概念

均线交叉策略是最经典的趋势跟踪策略之一。

**金叉：** 短期均线上穿长期均线 → 买入信号  
**死叉：** 短期均线下穿长期均线 → 卖出信号

---

## 🔧 策略实现

### 1. 策略基类

```python
from abc import ABC, abstractmethod
import pandas as pd

class Strategy(ABC):
    """策略基类"""
    
    @abstractmethod
    def generate_signals(self, df: pd.DataFrame) -> pd.Series:
        """生成交易信号：1=买入，-1=卖出，0=持有"""
        pass
    
    @abstractmethod
    def name(self) -> str:
        """策略名称"""
        pass
```

### 2. 均线交叉策略

```python
class MACrossStrategy(Strategy):
    """均线交叉策略"""
    
    def __init__(self, short_period: int = 5, long_period: int = 20):
        self.short_period = short_period
        self.long_period = long_period
    
    def name(self) -> str:
        return f"MA{self.short_period}/{self.long_period}金叉死叉"
    
    def generate_signals(self, df: pd.DataFrame) -> pd.Series:
        """生成交易信号"""
        signals = pd.Series(0, index=df.index)
        
        # 计算均线
        ma_short = df['收盘'].rolling(window=self.short_period).mean()
        ma_long = df['收盘'].rolling(window=self.long_period).mean()
        
        # 金叉：短期从下向上穿过长期
        golden_cross = (ma_short > ma_long) & (ma_short.shift(1) <= ma_long.shift(1))
        
        # 死叉：短期从上向下穿过长期
        death_cross = (ma_short < ma_long) & (ma_short.shift(1) >= ma_long.shift(1))
        
        # 生成信号
        signals[golden_cross] = 1
        signals[death_cross] = -1
        
        return signals
```

### 3. 双均线趋势策略

```python
class DualMAStrategy(Strategy):
    """双均线趋势策略（持仓信号，非脉冲式）"""
    
    def __init__(self, fast_period: int = 12, slow_period: int = 26):
        self.fast_period = fast_period
        self.slow_period = slow_period
    
    def name(self) -> str:
        return f"双均线{self.fast_period}/{self.slow_period}"
    
    def generate_signals(self, df: pd.DataFrame) -> pd.Series:
        """持仓信号：1=持有，0=空仓"""
        ma_fast = df['收盘'].rolling(window=self.fast_period).mean()
        ma_slow = df['收盘'].rolling(window=self.slow_period).mean()
        
        signals = pd.Series(0, index=df.index)
        signals[ma_fast > ma_slow] = 1
        
        return signals
```

## 📊 回测结果

**标的：** 平安银行 (000001)  
**时间：** 2024 年全年  
**初始资金：** 100,000 元  
**手续费率：** 0.03%（万分之三）

### 策略对比

| 策略 | 收益率 | 夏普比率 | 最大回撤 | 交易次数 | 胜率 |
|:---|:---|:---|:---|:---:|:---:|
| MA5/20 金叉死叉 | -1.81% | -0.24 | 14.61% | 20 | 30% |
| MA10/30 金叉死叉 | -3.15% | -0.33 | 11.60% | 10 | 60% |
| **双均线 12/26** | **+36.58%** | **1.25** | 12.12% | 2 | 100% |

### 关键发现

1. **金叉死叉策略交易频繁** - 手续费侵蚀利润
2. **双均线持仓策略更优** - 抓住大趋势，交易次数少
3. **参数敏感性** - 不同参数表现差异大
4. **2024 年平安银行趋势明显** - 适合趋势策略

---

## ⚠️ 注意事项

1. **震荡市表现差** - 均线策略在震荡市会反复止损
2. **手续费影响** - 频繁交易策略手续费成本高
3. **参数优化** - 需要针对不同股票优化参数
4. **配合其他指标** - 建议配合 RSI、MACD 等过滤信号
