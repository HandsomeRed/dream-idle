# Invest Analyzer 详细使用指南

## 概述

Invest Analyzer（投资分析器）是一个基于四流派投资体系的个股深度分析工具。它整合了 [sunheyi6/investTemplate](https://github.com/sunheyi6/investTemplate) V5.5.6 的核心方法论，提供标准化的投资分析流程。

## 四大投资流派

### 🛡️ 纯硬收息型

**特征**：
- 股息率 > 5%
- 连续5年以上派息记录
- 无有息负债或极低
- 经营现金流/净利润 > 100%
- 央国企背景优先

**适用标的**：电信、公用事业、REITs

**分析重点**：派息可持续性

### 🔍 价值发现型

**特征**：
- PE < 10 或 FCF/市值 > 10%
- 轻资产、商业模式清晰
- 周期拐点或低估
- 经营现金流充裕

**适用标的**：消费、制造、服务

**分析重点**：真实盈利能力

### 🚬 烟蒂股

**特征**：
- 净现金 > 市值
- PB < 0.3
- 靠近历史低点
- 无有息负债

**适用标的**：港股小盘、国企

**分析重点**：清算价值、管理人诚信

### 🔗 关联方资源型

**特征**：
- 国企背景
- 母公司资源可量化
- 控股平台折价

**适用标的**：物业、基建、公用事业

**分析重点**：利益输送风险、母公司支持力度

## 快速初筛清单

### 5分钟快速排除法

```
□ 地缘政治核查（30秒）
  - 非洲项目占比 > 10%？→ 排除
  - 东南亚冲突区业务？→ 排除

□ 价格位置检查
  - 距52周高点 < 30%？→ 排除（已反弹过高）

□ 负债快速扫描
  - 财务费用 > 5亿（非周期股）？→ 排除

□ 现金覆盖率
  - 现金 < 市值20%？→ 排除（安全边际不足）

□ 盈利连续性
  - 连续2年经营现金流为负？→ 排除
```

### 快速通过信号

- 🏆 财务费用/市值 < 50% → 无有息负债企业
- 🐢🍊 股价距52周低点 < 80% → 标准烟蒂股
- 🟠 价格位置不佳但估值合理 → 试探建仓 + 网格加仓

## FCF估值计算

### 核心公式

```
FCF = 经营现金流 - 维持性CAPEX
```

⚠️ **重要**：股息不扣除！股息是筹资活动现金流，不影响FCF计算。

### 剔除净现金FCF倍数

```
剔除净现金FCF倍数 = (市值 - 净现金) / FCF

其中：
- 净现金 = 现金及等价物 - 有息负债
- 企业价值(EV) = 市值 - 净现金
```

### 估值区间参考

| FCF倍数 | 评级 | 操作建议 |
|---------|------|---------|
| < 3倍 | 🐢🍊极品 | 极限仓位，重仓买入 |
| 3-5倍 | 🐢🍊金龟 | 推荐建仓区间 |
| 5-8倍 | 合理偏低 | 分批建仓 |
| 8-10倍 | 估值合理 | 持有/观望 |
| > 10倍 | 高估 | 卖出/回避 |

### FCF计算防坑指南

1. **股息不扣除**：FCF是经营+投资活动现金流，股息是筹资活动
2. **CAPEX要合理**：账面CAPEX可能含扩张性支出，需调整
3. **防双标**：不能"抵扣负债但不加现金"
4. **交叉验证**：PE和FCF两种方法都算，确保一致

## 试探建仓与网格加仓

### 建仓梯度

| 层级 | 仓位 | 条件 |
|------|------|------|
| 试探 | 3% | 初步通过初筛 |
| 首次 | 5% | 估值合理且价格可接受 |
| 二次 | 8% | 下跌10%后加仓 |
| 重仓 | 12% | 极限点位 |

### 持仓状态标签

- 🟢 正常：正常持有
- 🔵 已回本：股价回到成本价
- 🟡 高位：距低点涨幅>50%
- 🟠 关注：需要重点监控
- 🔴 遗留：历史遗留仓位

A: 直接说"帮我分析[股票代码/名称]"

### Q: 需要提供什么数据？
A: 股票代码或名称即可，我会尝试抓取公开数据，但建议提供关键财务数据以提高准确性

### Q: 分析结果可靠吗？
A: ⚠️ 所有AI分析必须经人工验证，数据以公司年报为准

### Q: 支持哪些市场？
A: 港股、A股为主，其他市场可参考方法论
