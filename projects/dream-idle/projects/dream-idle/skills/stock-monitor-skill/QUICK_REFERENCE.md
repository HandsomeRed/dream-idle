## 🎯 核心功能

### 七大预警规则
1. **成本百分比** - 盈利/亏损百分比提醒 ⭐⭐⭐
2. **日内涨跌幅** - 个股±4% / ETF±2% / 黄金±2.5% ⭐⭐
3. **成交量异动** - 放量>2 倍 / 缩量<0.5 倍 ⭐⭐
4. **均线金叉死叉** - MA5 上穿/下穿 MA10 ⭐⭐⭐
5. **RSI 超买超卖** - RSI>70 超买 / RSI<30 超卖 ⭐⭐
6. **跳空缺口** - 向上/向下跳空>1% ⭐⭐
7. **动态止盈** - 盈利 10%+ 后回撤 5%/10% ⭐⭐⭐

## ⚙️ 修改配置

### 配置文件位置
```
/root/.openclaw/workspace/skills/stock-monitor-skill/config.json
```

### 添加新股
在 `watchlist` 数组中添加：
```json
{
  "code": "000001",
  "name": "平安银行",
  "market": "sz",
  "type": "individual",
  "cost": 10.50,
  "alerts": {
    "cost_pct_above": 15.0,
    "cost_pct_below": -12.0,
    "change_pct_above": 4.0,
    "change_pct_below": -4.0
  }
}
```

### 修改成本价
找到对应股票，修改 `"cost"` 字段即可。

## 🔔 预警级别

- **🚨 紧急级** - 多条件共振 (3+ 指标)
- **⚠️ 警告级** - 2 个条件触发
- **📢 提醒级** - 单一条件触发

**防骚扰：** 同类预警 30 分钟内只发一次

---

## 💡 使用提示

✅ 多条件共振更可靠  
✅ 动态止盈保护利润  
✅ 技术指标确认趋势  
❌ 不要每个信号都操作  
❌ 单一指标容易假信号  

**核心原则：** "不错过大机会，不犯大错误"
