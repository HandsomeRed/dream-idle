## 2. FCF计算

### 核心公式

```
FCF = 经营现金流 - 维持性CAPEX
```

⚠️ **重要提醒**：
- 股息 **不扣除**（属于筹资活动现金流）
- CAPEX 应使用**维持性**而非账面值
- 需排除扩张性资本支出

### 计算示例

```
经营现金流：28.47亿
维持性CAPEX：4.0亿（合理估算）
────────────────
FCF：24.47亿 ✅
```

### CAPEX调整指南

| 行业 | 账面CAPEX特征 | 真实维持性判断 |
|------|-------------|---------------|
| 饮料/快消 | 线性折旧，偏高 | 约4亿/年 |
| 重工业 | 设备更新快 | 接近折旧额 |
| 科技 | 技术迭代快 | 可能高于折旧 |
| 公用事业 | 折旧大 | 需具体分析 |

---

## 3. 剔除净现金FCF倍数

### 公式

```
净现金 = 现金及等价物 - 有息负债
企业价值(EV) = 市值 - 净现金
剔除净现金FCF倍数 = EV / FCF
```

### 计算示例

```
市值：99亿
净现金：55亿（现金 - 有息负债）
EV = 99 - 55 = 44亿
FCF = 24.47亿
倍数 = 44 / 24.47 = 1.8倍 🐢🍊
```

---

## 4. 估值区间

### 剔除净现金FCF倍数

| 倍数区间 | 评级 | 操作建议 |
|---------|------|---------|
| < 3倍 | 🐢🍊 极品 | 极限仓位15%，重仓买入 |
| 3-5倍 | 🐢🍊 金龟 | 推荐建仓区间 |
| 5-8倍 | 合理偏低 | 分批建仓 |
| 8-10倍 | 估值合理 | 持有/观望 |
| > 10倍 | 高估 | 卖出/回避 |

### 传统PE（对比参考）

| PE区间 | 评级 | 操作建议 |
|--------|------|---------|
| < 6倍 | 低估 | 建仓 |
| 6-10倍 | 合理 | 持有 |
| 10-15倍 | 合理偏高 | 观望 |
| > 15倍 | 高估 | 卖出 |

## 5. 防坑检查

### ❌ 常见错误

1. **股息扣除**：FCF不用减股息！
2. **账面CAPEX**：需调整为维持性支出
3. **双标处理**：不能"抵扣负债但不加现金"
4. **PE陷阱**：利润质量差时禁用PE

### ✅ 正确做法

1. **方案A（推荐）**：不看净现金，纯FCF/市值
2. **方案B**：对称处理，加现金减负债
3. **两种方案都算**：确保估值逻辑一致
