# Invest Analyzer - 个股分析工具

> 基于四流派投资体系的个股深度分析工具

## ⚠️ 依赖要求

**本 skill 需要联网搜索能力！** 在分析股票前，必须先使用搜索工具获取财务数据。

### 必须安装
- **baidu-search** skill（首选搜索工具）
- 或 **agent-browser** skill（备选，用于抓取财经网站）

### 数据获取命令
```bash
# 使用 baidu-search（需要设置 BAIDU_API_KEY）
BAIDU_API_KEY="xxx" python3 skills/baidu-search/scripts/search.py '{"query":"股票名 2024年 财报", "search_recency_filter":"year"}'
```

## 简介

本技能整合自 [investTemplate V5.5.6](https://github.com/sunheyi6/investTemplate)，提供标准化的个股分析流程。

## 核心能力

- ✅ **5分钟快速初筛**：排除高风险标的
- ✅ **四流派判定**：纯硬收息 / 价值发现 / 烟蒂股 / 关联方资源
- ✅ **FCF估值计算**：剔除净现金FCF倍数核心指标
- ✅ **决策建议**：买入/观望/卖出 + 仓位建议
- ✅ **持仓管理**：状态标签、组合韧性评估

## 适用场景

```
用户：帮我分析[股票代码/名称]
    ↓
1. 联网搜索（使用 baidu-search 抓取年报数据）
2. 执行快速初筛（5分钟）
3. 判定投资流派
4. 深度财务分析
5. FCF估值计算
6. 输出决策建议
```

## 分析流程

### 第一步：快速初筛（5分钟）

| 检查项 | 排除标准 |
|--------|---------|
| 地缘政治 | 非洲/冲突区资产>10% ❌ |
| 价格位置 | 距52周高点<30% ❌ |
| 负债扫描 | 财务费用>5亿（非周期股）❌ |
| 现金覆盖 | 现金<市值20% ❌ |

### 第二步：流派判定

| 流派 | 特征 | 适用标的 |
|------|------|---------|
| 🛡️ 纯硬收息 | 股息率>5%，连续5年派息 | 电信/公用事业 |
| 🔍 价值发现 | PE<10，FCF/市值>10% | 消费/制造 |
| 🚬 烟蒂股 | 净现金>市值，PB<0.3 | 港股小盘 |
| 🔗 关联方 | 国企背景、母公司支持 | 物业/基建 |

### 第三步：FCF估值（核心）

```
FCF = 经营现金流 - 维持性CAPEX
（注意：股息不扣除！）

剔除净现金FCF倍数 = (市值 - 净现金) / FCF

估值参考：
- 3-5倍：金龟级（推荐建仓）
- 5-8倍：合理偏低
- 8-10倍：估值合理
- >10倍：高估
```

### 第四步：决策输出

```
输出格式：
- 标的：[名称/代码]
- 流派：[判定结果]
- 初筛结果：[通过/未通过]
- 核心指标：[FCF/FCF倍数/股息率等]
- 估值结论：[低估/合理/高估]
- 建议：[买入/观望/卖出]
- 仓位建议：[试探/建仓/加仓/清仓]
- 风险提示：[关键风险点]
```
