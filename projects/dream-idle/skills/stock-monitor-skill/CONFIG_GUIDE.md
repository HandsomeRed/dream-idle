## 🎯 七大预警规则

### 1. 成本百分比预警 ⭐⭐⭐
- 基于持仓成本计算盈亏百分比
- 达到阈值自动提醒

### 2. 日内涨跌幅预警 ⭐⭐
- 个股：±4%
- ETF：±2%
- 黄金：±2.5%

### 3. 成交量异动 ⭐⭐
- 放量 > 2 倍 5 日均量
- 缩量 < 0.5 倍 5 日均量

### 4. 均线金叉/死叉 ⭐⭐⭐
- MA5 上穿 MA10 → 金叉
- MA5 下穿 MA10 → 死叉

### 5. RSI 超买超卖 ⭐⭐
- RSI > 70 → 超买
- RSI < 30 → 超卖

### 6. 跳空缺口 ⭐⭐
- 向上/向下跳空 > 1%

### 7. 动态止盈 ⭐⭐⭐
- 盈利 10%+ 后回撤 5% → 减仓提醒
- 盈利 10%+ 后回撤 10% → 清仓提醒

## ⚙️ 修改配置

### 编辑配置文件

配置文件位于：`/root/.openclaw/workspace/skills/stock-monitor-skill/config.json`

#### 添加新股票

```json
{
  "code": "000001",
  "name": "平安银行",
  "market": "sz",
  "type": "individual",
  "cost": 10.50,
  "alerts": {
    "cost_pct_above": 15.0,
    "cost_pct_below": -12.0,
    "change_pct_above": 4.0,
    "change_pct_below": -4.0,
    "volume_surge": 2.0,
    "ma_monitor": true,
    "rsi_monitor": true,
    "gap_monitor": true,
    "trailing_stop": true
  }
}
```

#### 修改成本价

找到对应股票，修改 `"cost"` 字段：
```json
"cost": 55.50  // 改成你的实际成本价
```

#### 调整预警阈值

```json
"alerts": {
  "cost_pct_above": 20.0,     // 盈利 20% 提醒
  "cost_pct_below": -15.0,    // 亏损 15% 提醒
  "change_pct_above": 5.0,    // 日内异动±5%
  "volume_surge": 3.0         // 放量 3 倍提醒
}
```

#### 开关技术指标

```json
"alerts": {
  "ma_monitor": false,     // 关闭均线监控
  "rsi_monitor": true,     // 开启 RSI 监控
  "gap_monitor": true,     // 开启跳空监控
  "trailing_stop": false   // 关闭动态止盈
}
```

## ⚠️ 使用提示

1. **技术指标有滞后性**：均线、MACD 等都是滞后指标，用于确认趋势而非预测
2. **避免过度交易**：预警只是参考，不要每个信号都操作
3. **多条件共振更可靠**：单一指标容易假信号，多条件共振更准确
4. **动态止盈要灵活**：回撤 5% 减仓、10% 清仓是建议，根据市场灵活调整
5. **防骚扰机制**：同类预警 30 分钟内只发送一次
