### 1. 伤害公式类型

#### 减法公式（梦幻西游采用）
```
伤害 = (攻击 - 防御) × 系数
```
**特点：**
- 攻防差值决定伤害
- 防御属性非常重要
- 代表作：《梦幻西游》《神武》《问道》

#### 除法公式
```
伤害 = 攻击 / (攻击 + 防御) × 系数
```
**特点：**
- 防御收益递减
- 代表作：《天龙八部》

#### 乘法公式
```
伤害 = 攻击 × (1 - 免伤率)
```
**特点：**
- 无防御属性，用免伤率
- 代表作：《大话西游》

### 2. 梦幻西游法术伤害公式

#### 通用公式
```
法术伤害 = (师门技能项 + 法伤 - 法防) × 阵法 × 分灵系数 × (1 + 修炼差 × 0.02) + 修炼差 × 5 + 神木符
```

#### 简化版（我们的游戏）
```
法术伤害 = (师门技能项 + 法伤 - 法防) × 分灵系数
```

### 3. 师门技能项公式（各门派）

#### 龙宫 - 龙卷雨击
```
师门技能项 = 技能等级² / 145 + 技能等级 × 1.4 + 35
```

#### 神木林 - 落叶萧萧
```
师门技能项 = 技能等级² / 120 + 技能等级 × 1.5 + 30
```

#### 魔王寨 - 三昧真火
```
师门技能项 = 技能等级² / 100 + 技能等级 × 2.3
```

#### 化生寺 - 唧唧歪歪
```
师门技能项 = 技能等级 × 1
```

### 4. 分灵系数（群体法术）

```
分灵系数 = 1 - 目标个数 × 0.1
下限：0.5（最多秒 5 个）
```

**示例：**
- 单体：1.0
- 秒 3：0.7
- 秒 5：0.5

---

### 5. 物理攻击公式

#### 普通攻击
```
伤害基数 = 攻击 - 防御
```

#### 暴击
```
伤害基数 = (攻击 - 防御) × 2
```

#### 连击
```
第一次：(75% × 攻击 - 防御) × 2
第二次：(75% × 攻击 - 防御) × 2
```

#### 实际伤害修正
| 场景 | 修正系数 |
|:---|:---|
| 人 VS 人 | × 1.0 |
| 人 VS 宠 | × 0.9 |
| 宠 VS 人 | × 0.8 |
| 宠 VS 宠 | × 1.0 |

---

### 6. 属性成长公式

#### 血量成长
```
HP = 基础 HP + 等级 × HP 成长 + 体质点数 × 每点 HP
```

#### 魔法成长
```
MP = 基础 MP + 等级 × MP 成长 + 魔力点数 × 每点 MP
```

#### 攻击成长
```
攻击 = 基础攻击 + 等级 × 攻击成长 + 力量点数 × 每点攻击
```

---

### 7. 升级经验曲线

#### 常用公式
```
EXP(等级) = A × 等级^B + C
```

**参数示例：**
- A = 100, B = 2.5, C = 0
- 1 级：100 EXP
- 10 级：31,623 EXP
- 50 级：1,767,767 EXP
- 100 级：10,000,000 EXP

#### 线性增长
```
EXP(下一级) = 当前 EXP × 增长率
增长率：1.1 ~ 1.3
```

### 战斗系统

#### 伤害计算
```javascript
// 物理伤害
function calcPhysicalDamage(atk, def, isCrit = false) {
  let base = atk - def;
  if (isCrit) base *= 2;
  return Math.max(1, Math.floor(base));
}

// 法术伤害
function calcMagicalDamage(skillLevel, mag, res, targetCount = 1) {
  const skillTerm = Math.pow(skillLevel, 2) / 120 + skillLevel * 1.5 + 30;
  const splitFactor = Math.max(0.5, 1 - targetCount * 0.1);
  const base = (skillTerm + mag - res) * splitFactor;
  return Math.max(1, Math.floor(base));
}
```

#### 出手顺序
```javascript
// 速度决定出手顺序
function getTurnOrder(combatants) {
  return combatants.sort((a, b) => b.speed - a.speed);
}
```
