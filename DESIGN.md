## 🎯 核心设计理念

**梦幻西游的本质 → 放置挂机转化**

| 梦幻西游 | 放置化改造 | 说明 |
|:---|:---|:---|
| 师门任务 | 每日师门（一键完成） | 按修为/等级给奖励 |
| 押镖 | 离线挂机收益 | 在线=押镖中，按战力计算 |
| 捉鬼 | 自动捉鬼副本 | 设置次数，离线也捉 |
| 副本 | 副本扫荡 | 通关后可扫荡，消耗体力 |
| 剧情任务 | 主线推图 | 关卡制，星级评价 |
| 帮派修炼 | 修炼系统 | 消耗资源升级属性 |
| 装备打造 | 装备 + 特效 | 随机属性 + 套装效果 |
| 宠物 | 伙伴/宠物系统 | 助战 + 收集养成 |
| 技能 | 技能搭配 | 主动 + 被动，可升级 |
| ~~社交~~ | ❌ 删除 | 单机游戏不需要 |

---

## 🔄 核心循环

```

     ↓
  离线挂机收益（时间换资源）
```

## 💰 经济系统

### 资源类型
| 资源 | 用途 | 获取方式 |
|:---|:---|:---|
| 金币 | 基础消耗 | 推图、离线收益 |
| 银币 | 高级消耗 | 副本、活动 |
| 体力 | 副本次数 | 自然恢复、购买 |
| 装备石 | 装备强化 | 分解、商店 |
| 技能书 | 技能升级 | 掉落、兑换 |
| 宠物粮 | 宠物养成 | 掉落、活动 |

### 消耗循环
```
推图 → 金币/装备 → 强化角色 → 推更远
  ↓
副本 → 银币/技能书 → 升级技能 → 战力提升
  ↓
离线 → 资源累积 → 集中养成 → 次日继续
```

## 📈 数值设计

### 战力计算
```
战力 = (攻击 × 2 + 防御 × 1.5 + 气血 × 0.5 + 速度 × 1) × 技能系数 × 装备系数
```

### 推图难度
- 1-20 关：新手引导，轻松通过
- 21-50 关：需要适当养成
- 51-80 关：卡点设计，需要离线积累
- 81-100 关：深度养成验证

### 离线收益
```
每小时收益 = 基础收益 × (1 + 推图关卡数 × 0.1) × 战力系数
离线上限 = 12 小时（防止囤积）
```
