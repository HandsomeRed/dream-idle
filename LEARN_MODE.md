# 🚀 学习任务驱动模式

**生效时间：** 2026-03-12 23:53  
**模式：** Task-Driven (任务驱动)

---

## 📋 核心原则

**完成一个小任务 → 休息 5 分钟 → 继续下一个任务**

---

## ⏱️ 学习节奏

```
┌─────────────────┐    ┌──────────┐    ┌─────────────────┐
│   学习任务 #1   │ →  │ 休息 5 分钟 │ →  │   学习任务 #2   │
│   (25-55 分钟)   │    │          │    │   (25-55 分钟)   │
└─────────────────┘    └──────────┘    └─────────────────┘
       ↓                                      ↓
   功能实现 + 测试                         功能实现 + 测试
       ↓                                      ↓
   提交代码                              提交代码
```

## ✅ 优势

1. **更灵活** - 不受固定时长限制，按任务完成度推进
2. **更高效** - 小步快跑，快速迭代
3. **更可持续** - 5 分钟短暂休息，保持专注
4. **更可追踪** - 每个任务都有明确开始/结束时间

---

## 📝 使用说明

### 自动模式（未来实现）

```
心跳检查 (每 5 分钟)
  ↓
检查当前任务状态
  ↓
如果任务完成 → 进入 5 分钟休息
  ↓
休息结束 → 自动开始下一个任务
```

### 当前模式（人工驱动）

```
用户触发任务 → 执行 → 完成 → 休息 5 分钟 → 用户触发下一个任务
```
