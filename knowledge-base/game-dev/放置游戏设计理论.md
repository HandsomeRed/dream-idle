## 📖 核心概念

放置挂机游戏（Idle Game）的核心设计原则，参考梦幻西游题材改编。

### 四大核心要素

1. **反馈循环** - 立即反馈激励玩家
   - 每次点击/操作都有视觉和数值反馈
   - 收益数字跳动、特效展示
   - 短周期（秒级）反馈循环

2. **增量进展** - 不断增加收益速率
   - 属性成长曲线
   - 解锁新功能
   - 数值膨胀但可控

3. **简单控制** - 任何玩家都能上手
   - 单手可操作
   - 界面简洁
   - 自动战斗为主

4. **自动化收益** - 离线也能获得进度
   - 离线收益计算
   - 自动寻路任务
   - 后台挂机

| 原作机制 | 放置改编 | 实现方式 |
|:---|:---|:---|
| 回合制战斗 | 自动战斗 | 每秒自动攻击 |

| 社交交互 | 异步交互 + 排行榜 | 离线可见 |
| 经济系统 | 自动产出 | 定时结算 |

## 📊 数值设计原则

### 成长曲线

```
收益 = 基础值 × (1 + 加成系数)^时间
```

### 阶段划分

- **前期（0-1 小时）：** 快速反馈，每秒都有收益
- **中期（1-10 小时）：** 解锁新功能，保持新鲜感
- **后期（10+ 小时）：** 深度养成，追求极限

---

## ⚠️ 注意事项

1. **避免数值崩坏** - 设置合理的成长上限
2. **保持新鲜感** - 定期解锁新功能
3. **离线收益平衡** - 不能太高（失去动力）也不能太低（挫败感）
4. **视觉反馈** - 数字跳动、特效、音效
