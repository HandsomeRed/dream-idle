### 2. 回测引擎实现

```python
class Backtester:
    """回测引擎"""
    
    def __init__(self, initial_capital: float = 100000, 
                 commission_rate: float = 0.0003):
        """
        初始化回测引擎
        
        Args:
            initial_capital: 初始资金（默认 10 万）
            commission_rate: 手续费率（默认万分之三）
        """
        self.initial_capital = initial_capital
        self.commission_rate = commission_rate
    
    def run(self, df: pd.DataFrame, strategy: Strategy) -> Dict:
        """运行回测"""
        # 1. 生成信号
        signals = strategy.generate_signals(df)
        
        # 2. 初始化
        capital = self.initial_capital
        position = 0
        trades = []
        
        # 3. 遍历数据
        for i in range(len(df)):
            row = df.iloc[i]
            signal = signals.iloc[i]
            price = row['收盘']
            
            # 买入信号
            if signal == 1 and capital > 0:
                shares = int(capital / (price * 100)) * 100
                if shares > 0:
                    cost = shares * price * (1 + self.commission_rate)
                    capital -= cost
                    position = shares
            
            # 卖出信号
            elif signal == -1 and position > 0:
                revenue = position * price * (1 - self.commission_rate)
                capital += revenue
                position = 0
        
        # 4. 返回结果
        return {
            'initial_capital': self.initial_capital,
            'final_value': capital,
            'total_return': (capital - self.initial_capital) / self.initial_capital * 100,
            'trades': trades
        }
```

## 📊 绩效指标

### 1. 总收益率

```python
total_return = (final_value - initial_capital) / initial_capital * 100
```

### 2. 年化收益率

```python
annual_return = (1 + total_return/100) ** (252/days) - 1
```

### 3. 夏普比率

```python
# 夏普比率 = (年化收益率 - 无风险利率) / 年化波动率
sharpe_ratio = (annual_return - 0.03) / (volatility/100)
```

### 4. 最大回撤

```python
# 最大回撤 = max(历史最高值 - 当前值) / 历史最高值
max_drawdown = max((peak - trough) / peak * 100)
```

### 5. 胜率

```python
# 胜率 = 盈利交易次数 / 总交易次数
win_rate = profitable_trades / total_trades * 100
```

## ⚠️ 注意事项

1. **未来函数** - 避免使用未来数据（如用收盘价决定当日买卖）
2. **手续费** - 必须考虑手续费，否则结果过于乐观
3. **滑点** - 实际成交价可能与信号价不同
4. **过拟合** - 参数优化要在样本外验证
5. **幸存者偏差** - 使用当时存在的股票列表
